# unix-time-converter
A simple Python CLI tool for converting between Unix timestamps and human-readable date formats.

## Features
- Convert a standard date (YYYY-MM-DD HH:MM:SS) to a Unix timestamp.
- Convert a Unix timestamp back to a human-readable date.
- Easy-to-use command-line interface.
